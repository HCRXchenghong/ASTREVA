# 星渡官网自研后台闭环

## 本地入口

- 前台：`http://127.0.0.1:4321/`
- 后台登录：`http://127.0.0.1:1337/admin-2fa`
- 官网后台：`http://127.0.0.1:1337/site-admin/`
- 内容接口：`http://127.0.0.1:1337/site-admin-api/public-content`
- 重建服务：`http://127.0.0.1:8787/healthz`
- 客服用户端：`http://127.0.0.1:5173/?embed=1`

## 登录流程

首次进入后台时创建管理员邮箱和密码，然后用 Microsoft Authenticator 扫描二维码绑定 2FA。后续每次登录都需要邮箱、密码和 2FA 验证码。

## 内容闭环

自研后台将官网内容保存到 `data/site-content.json`，客户咨询保存到 `data/leads.json`，上传文件保存到 `uploads/site-admin/`。

后台可编辑范围：

- 全站设置：品牌、Logo、电话、邮箱、导航、页脚、社交、Cookie、客服地址、默认 SEO。
- 首页：首屏文字、背景图、背景视频、播放视频、合作伙伴、产品介绍、优势、评价、CTA。
- 产品中心：产品中心首屏、品类页文案、详情页文案。
- 品类与产品：支持新增、编辑、删除；产品支持图库、视频、卖点、规格、下载资料、SEO。
- 服务与静态页：服务支持、关于我们、质保、安装指南、FAQ、隐私政策、用户协议。
- 客户咨询：查看表单提交，维护状态、优先级和内部备注。
- 媒体库：上传图片、视频、PDF、ZIP、CAD 文件并回填 URL。

## 发布流程

1. 在后台右侧表单修改内容。
2. 点击“保存当前模块”或顶部“保存草稿”。
3. 点击“发布官网按钮”。
4. 后台请求 `http://127.0.0.1:8787/rebuild`，重新构建 Astro 静态页面。
5. 构建成功后刷新前台预览。

前台构建时直接读取自研后台的 `site-admin-api/public-content`，生成纯静态 HTML；后台服务未启动时会回退到内置种子数据，保证本地开发不白屏。
